# ⚔️ Data Quest: The AI Engineer RPG

A premium, interactive browser-based role-playing game engineered using strict **Object-Oriented Programming (OOP)** principles. This technical portfolio piece gamifies data science workflows—such as data cleaning, model training, and performance tuning—into automated, turn-based battle mechanics and state mutations without depending on heavy third-party gaming frameworks.

## 🚀 Core Architectural Capabilities

* **Object-Oriented State Engine:** Utilizes isolated JavaScript ES6 classes to encapsulate player stats, computational energy levels, and real-time calculation thresholds.
* **Turn-Based Combat Array:** Translates data engineering tasks into strategic pipeline choices (`.fillna()` parsing vs. complex Random Forest ensemble processing).
* **Asynchronous Console Terminal:** Features a custom log architecture that simulates live execution traces, streaming pipeline logs dynamically.
* **Responsive Matrix Layout:** Styled completely from scratch with a dark developer aesthetic using Tailwind CSS and dynamic progress bar animations.

---

## 🛠️ Technology Stack & Dependencies

* **Core Engine:** Vanilla JavaScript (ES6+ Classes & Prototypical Inheritance)
* **Design Frame:** Tailwind CSS (Fluid HUD Containers & Adaptive Typography)
* **Storage Layer:** Native Web API Controllers
